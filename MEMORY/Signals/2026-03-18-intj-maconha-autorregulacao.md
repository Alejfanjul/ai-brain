# INTJ x Maconha x Autorregulação — 18/03/2026

## Contexto

Após 5 dias consecutivos de uso (13-17 Mar), dia 2 sem fumar. Relatando tristeza, falta de vontade, olhar negativo pro trabalho. Reconhece que é rebote químico mas está lutando contra a racionalização pra fumar de novo.

## Padrão INTJ identificado

1. **Alívio do overthinking:** Ni dominante = mente que não desliga. Maconha funciona como botão de "off" — atalho pro Se inferior (experiência sensorial/presente).

2. **Racionalização motivada:** Ni-Te = hardware perfeito pra construir argumentos lógicos convincentes a favor do impulso. A "história na cabeça" antes de fumar é o estrategista trabalhando a favor do uso. A decisão já foi tomada emocionalmente — a história é o advogado de defesa.

3. **Paradoxo do controle:** INTJs valorizam controle. Perder controle do uso gera dissonância que oscila entre "preciso parar 100%" (perfeccionismo) e "eu escolho, eu controlo" (racionalização). Os dois extremos alimentam o ciclo.

4. **Perfeccionismo → frustração → uso:** Ni cria visão clara de como as coisas deveriam ser. Gap entre visão e realidade (negócio, treino) gera frustração. Maconha alivia temporariamente.

5. **Bypass emocional:** Fi terciário = sente profundamente mas processa emoções com dificuldade. Maconha faz bypass. Emoção não processada volta — e volta pior (rebote).

## Insight central

Maconha não é o problema — é a **solução atual** para necessidades reais (desligar mente, sentir presente, aliviar frustração). O caminho é encontrar alternativas que atendam as mesmas necessidades sem o rebote (treino pesado, atividade sensorial intensa).

## Contra-medidas para racionalização

- **Regra mecânica sem negociação:** max 2x/semana + nunca consecutivo. Não é sugestão, é protocolo.
- **Reconhecer a história como sintoma:** quando está construindo argumento pra fumar, isso em si é o sinal de que não deve.
- **Delay de 30 minutos:** esperar a urgência passar antes de decidir.
