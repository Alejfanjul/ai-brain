- **install.sh guiado** — parte WSL automática, parte Windows manual com instruções claras
- **Autostart via VBScript** — roda minimizado, sem janela visível

## Learnings

- Distro WSL pode ter nome diferente do esperado (`Ubuntu-22.04` vs `Ubuntu`) — verificar com `wsl -l -v`
- Python no Windows: `pip` frequentemente não está no PATH mesmo com Python instalado — sempre usar `py -X.Y -m pip`
- Caminhos WSL via PowerShell (`\\wsl.localhost\`) são instáveis — copiar pelo lado WSL via `/mnt/c/`
- PowerShell quebra comandos colados com quebra de linha — instruir a colar numa linha só
